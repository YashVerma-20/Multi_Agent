# System Rules: Meta-Agentic OS

- **Backend**: FastAPI with `asyncio`. Use `Pydantic v2`.
- **Database**: `aiosqlite` for local dev.
- **LLM Integration**: 
  - `google-generativeai` for Gemini 1.5 Flash/3.1.
  - `groq` for Llama-3 (Fast Chat).
- **Frontend**: React (Vite) + Tailwind CSS. 
- **UI Style**: Claude.ai clone. Use `#F9F9F8` for background, `#111827` for text. High whitespace.
- **Authentication**: Simple JWT-based auth (placeholder for now).
